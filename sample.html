HTML

HTML stands for Hyper Text Markup Language.

Currently, we are using the latest version of HTML 5.2. HTML is used for creating webpages.

Based on HTML elements the browser displays the content such as "This is a Heading", "This is a Paragraph", "This is a link" etc.We have to save the html code with extension ".html" or ".htm".

An HTML element is defined by a start tag, some content, and an end tag.
